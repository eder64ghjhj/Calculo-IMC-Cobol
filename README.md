# Calculadora de IMC em COBOL 🚀

Este projeto é um utilitário simples desenvolvido em COBOL para calcular o Índice de Massa Corporal (IMC). 

## 💻 Sobre o Código
O programa solicita a entrada de peso e altura, processa o cálculo matemático e exibe o resultado formatado na tela. 

### 🛠️ Desenvolvimento e Validação
O código foi estruturado seguindo as divisões padrão da linguagem (Identification, Data e Procedure). 
* **Compilador Utilizado:** GnuCOBOL.
* **Validação:** Devido a particularidades de ambiente local no Windows (bibliotecas GMP), a execução e a lógica foram validadas e testadas via compilador online (JDoodle).

## 🚀 Como Executar
1. Copie o conteúdo do arquivo `IMC.cbl`.
2. Utilize um compilador COBOL (local ou online) para processar o arquivo.
3. Insira os dados conforme solicitado pelo programa.
